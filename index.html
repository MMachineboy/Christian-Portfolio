How to create Your Own Website

Creating my own website with HTML, CSS, and a bit of JavaScript has been both challenging and rewarding. HTML gave me the structure-the backbone of the site- while CSS allowed me to bring it to life with colors, layouts and a style that actually reflects with who I am. Onced I added JavaScript, the site stopped feeling static and started feeling interactive, whether it was simple button actions or small dynamic features that improve the user experience. Building everything from stratch helped me understand how the web really works behind the scenes, not just how things look on the surface. More than anything, it gave me confidence: knowing that I can take an idea and turn it into a functional website using code I wrote myself is what motivates me to keep learning and improving.
